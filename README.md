# "NeWork"

## Дипломная работа курса Нетологии "Android-разработчик с нуля".

### Краткое описание.

Приложение в котором пользователи
могут создавать посты и события с медиафайлами и отмечать их на Яндекс картах, указывать места
работы и социальные связи (упоминание в постах, конференциях).

### Инструменты.

- Архитектура MVVM
- Библиотеки:
    - Material Design
    - ROOM
    - OKHTTP
    - Retrofit
    - Hilt
    - LiveData, Flow
    - Coroutines
    - YandexMapsMobile
    - [ImagePicker](https://github.com/Dhaval2404/ImagePicker)

### Функционал приложения.

- **Основной экран.**
     Содержит экран постов , событий и пользователей, а также меню с авторизацией или профилем в случае если пользователь авторизирован
